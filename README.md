### 🎨 تصميم احترافي
- تصميم فاخر باللون الذهبي والأسود
- تأثيرات حركية سلسة باستخدام Framer Motion
- Responsive كامل على جميع الأجهزة
- خطوط عربية وأجنبية احترافية

### 🛒 سلة تسوق ذكية
- إضافة/حذف/تعديل الكميات
- حساب تلقائي للإجمالي
- drawer جانبي أنيق
- إتمام الطلب عبر واتساب

### 📱 تجربة مستخدم سلسة
- تنقل سلس بين الأقسام
- تصفية المنتجات حسب الفئة
- تأثيرات hover جميلة
- تحميل سريع

### 🔍 SEO جاهز
- Metadata مُحسّنة
- Open Graph tags
- Structured data
- Performance optimized

## 🚀 التشغيل

### 1. تثبيت الحزم
```bash
npm install
```

### 2. تشغيل في وضع التطوير
```bash
npm run dev
```

### 3. البناء للإنتاج
```bash
npm run build
```

### 4. تشغيل الإنتاج
```bash
npm start
```
